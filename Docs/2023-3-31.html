微软推送Moment3


截至2023年2月28日16:48分,微软已向正式用户开始推送更新Moment3

该更新为不仅开始菜单添加了一个与当前配色方案匹配的搜索框,而且还增加了一个功能:"Microsoft账户的通知"
“Microsoft 账户的通知”会在 Windows 11 开始菜单的注销弹出窗口中显示，用于推广微软的产品和服务，例如 OneDrive 和 Microsoft 365。这一功能目前只对少数用户开放，未来几个月将逐步扩大范围。
这次预览更新老掉牙的微软还给包含了许多改进和错误修复，增加了 Microsoft Defender for Endpoint 的新功能，解决了 Microsoft PowerPoint (PPT)冻结的问题。
微软警告，从下个月开始，不再有Windows10 的支持版本20H2和Windows 10的支持版本 21H2 的可选、非安全预览版本，只有累积性月度安全更新(B或星期二版本),Windows10版本22H2将继续接收安全和可选版本更新。
